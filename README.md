# Weekly Report Skill

Agent skill that generates and sends weekly reports from GitHub, Slack, and Notion. Delivers via Email, LINE, and LinkedIn DM. Includes Q&A auto-check loop.

## Install

Upload `weekly-report.skill` via Claude Desktop's "Upload a skill" GUI.

Or clone and use directly:

```bash
git clone https://github.com/ryanlinjui/agent-backend-weekly-report
```

## Usage

- `weekly report` / `週報` — generate + send report
- `qa` / `check replies` / `回覆` — check + reply to questions
