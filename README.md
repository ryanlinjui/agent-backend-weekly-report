# Weekly Report Skill

Agent skill that generates and sends weekly reports from GitHub, Slack, and Notion. Delivers via Email, LINE, and LinkedIn DM.

## Install

```bash
git clone https://github.com/ryanlinjui/agent-backend-weekly-report
```

Open in Claude Code or drag folder into Claude Desktop.

## Usage

- `weekly-report` — generate + send report
- `weekly-report-init` — first-time setup  
- `weekly-report-qa` — check + reply to questions
